# SEO OS Dashboard Design Reference

Captured from the dashboard design reference used for the SEO OS prototype.

## Reference description

A sleek analytics dashboard interface with polished, data-driven business management styling. The design shows widgets for:

- Sales
- Expenses
- Customer satisfaction
- Bar chart analytics
- AI insights or intelligent recommendations

## Visual direction to preserve

- Modern SaaS analytics dashboard feel
- Clean card-based layout
- Strong visual hierarchy
- Business-management dashboard, not a generic chat interface
- Data-rich but polished and easy to scan
- Widgets should feel actionable, with AI insights layered into the dashboard
- Suitable direction for the SEO OS dashboard UI

## How to apply to `this template`

Use this as a UI reference when refining the SEO OS dashboard:

- Replace generic panels with high-quality analytics cards.
- Include executive-level KPI widgets for SEO performance, opportunities, approvals, and agent activity.
- Add AI insight cards that explain what changed, why it matters, and what action is recommended.
- Use a polished dashboard layout similar to a business intelligence product.
- Keep the interface useful for client/site onboarding, work tracking, data, and approvals.

## Suggested SEO OS widget equivalents

| Reference widget | SEO OS equivalent |
|---|---|
| Sales | Organic leads, conversions, revenue proxy, traffic value |
| Expenses | SEO task cost, content production effort, tool/API usage |
| Customer satisfaction | Client health, approval response rate, site confidence score |
| Bar chart widgets | GSC clicks/impressions, keyword movement, content velocity |
| AI insights | Loki/SEO agent recommendations, risks, next best actions |
